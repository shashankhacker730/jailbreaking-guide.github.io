Checkra1n

This method only supports A7-A11 and iOS 12.3 and up. Checkra1n currently lacks support for:
  • iPad Air 2
  • iPad (5th Gen)
  • iPad Pro (1st Gen)
    • Before proceeding with this guide, there are some things you will need to consider;
    • This is a beta, so use at your own risk
    • This release is MacOS only, Windows and Linux support are still being worked on
    • Cydia is the package manager that can be installed, support for others will be added

    For this guide you will need the following;
    • Checkra1n
    • macOS or Linux (Linux guide not done)
    • A lightning adapter to connect to your computer
    Step 1: Checkra1n Application
    1. After downloading the latest Checkra1n, move the application from the DMG folder to the Applications folder
    2. Plug your iDevice into your computer and run the Checkra1n application
    Step 2: Using Checkra1n
    Follow the on-screen instructions that the Checkra1n application displays.
    Step 3: Checkra1n Installer
    1. After your device resprings, you should the Checkra1n Installer installed on your Home Screen
    2. Launch the new checkra1n app and it will show a list of package managers, select the package manager you desire and it should install
    3. Launch the package manger and let it (re)load its packages. If you are prompted for an upgrade, tap Upgrade All
    Once all upgrades are completed and you've resprung, you are free to use any (compatible) jailbreak tweaks. Also feel free to check out the Jailbreaking Essentials!
    If your package manager and/or any tweaks you have aren't working check Troubleshooting page. Make sure the tweak is compatible for your ios version before hand.
