Sideloading Applications

Cydia Impactor and many signing services have been having issues. Here is how to sideload apps.
Cydia Impactor
For this guide you will need the following;
  • Cydia Impactor
  1. Download Cydia Impactor and extract it to your desktop
  2. Run impactor and drag and drop the ipa file you want to sideload
  3. Put in your Apple ID credentials and it will start side loading the application
  4. Check your device and trust the profile

Reprovision
For this guide you will need the following;
  • Matchstic repo
  1. Add the Matchstic Repo to cydia, or your perferred package manager
  2. Search for Reprovision and install it via cydia
  3. Launch Reprovision from your springboard and type in your Apple ID Credentials.
