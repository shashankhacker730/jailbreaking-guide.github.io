OpenSSH

This method supports all jailbroken devices provided you have an WiFi connection.
Step 1: Install SSH
This guide assumes you are already jailbroken. To find a jailbreak suitable for your device, click here.
  • If you're running unc0ver you can install SSH by activating the (Re)Install OpenSSH toggle and rejailbreaking.
  • OpenSSH can be installed via the default repos on other jailbreaks.
Step 2: Getting Device IP
SSH is a peer-to-peer, secure shell protocol. To access a device over SSH/SFTP, you will need the device's IP, a username, and password.
  1. Open Settings and connect to the same WiFi network your computer is connected to
  2. Once you are connected, tap the next to your connected WiFi network and take note of the IP Address
Step 3: SSH Client
Now that you have your iOS Device's IP Address, you can go ahead use an SSH Client to access it.
Accessing a device over SSH is slightly different across operating systems. Please follow the directions according to your OS;
Windows

File Transfer (SFTP):

  1. Download and install Filezilla Client
  2. At the top of the window, in the Host: field, input your iOS Device IP
  3. In the Username: field, input root
  4. In the Password: field, input alpine
  5. Check to make sure all inputted credentials is correct and click Quickconnect

Command Line Interface (SSH)

  1. Download and install PUTTY or your choice of SSH Client
  2. In the Host Name (or IP Address) field, input your iOS Device IP
  3. Select the SSH connection type and click Open
macOS / Linux / Unix / FreeBSD -- Command Line Interface (SSH)
  1. Open Terminal and type ssh root@XXX.XXX.XXX.XXX and press Enter (replace X with your iOS Device IP)
macOS / Linux -- File Transfer (SFTP)
  1. Download and install Filezilla Client
  2. At the top of the window, in the Host: field, input your iOS Device IP
  3. In the Username: field, input root
  4. In the Password: field, input alpine
  5. Check to make sure all inputted credentials is correct and click Quickconnect
If the connection attempt fails you may A) Have changed the default password B) Not inputted credentials correctly or C) Not be connected to the same WiFi network
