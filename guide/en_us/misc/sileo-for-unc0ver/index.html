Sileo for Unc0ver

This method supports devices running iOS 11.0 - 12.4 on the Unc0ver Jailbreak. If you happen to be using Chimera, but want to use Cydia, you're going to have to jailbreak using Unc0ver.
Step 1: Add Diatrus APT Repo
  1. Open your favored package manager and add https://diatr.us/sileo/
Step 2: Install DiatrusInstaller
  1. Download & Install Diatrus Installer
Step 3: Install Sileo
  1. Open DiatrusInstaller
  2. Tap Install and wait for the conformation message
Once you have received the conformation popup, you're now ready to use Sileo. If the installation fails, reboot and rejailbreak with Reinstall Cydia turned on.
