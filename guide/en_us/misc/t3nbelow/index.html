T3nBelow

This is a downgrade for 10.x / 9.x to 8.4.1. Supports iPhone 4s, iPhone 5, and iPod Touch 5G
Before continuing this guide, please make sure you have backed up your iOS Device via iTunes and/or iCloud.
Only have this and default packages installed via cydia. Remove any other tweaks you have installed.
For this guide you will need the following;
  • Jailbroken with cydia access
Step 1: Getting T3nBelow
  1. Download and install this deb file using Filza (or any other method for install debs)
  2. After you've installed T3nbelow, return home and you should see the app "T3nBelow" on your homescreen.
  3. Launch the app and continue to step 2.
Step 2: Downgrading
  1. Press the downgrade button and select your device when asked.
    2. Make sure you select the correct device to get the best results
  2. When it is done click reboot. The device will reboot and "update".
  3. Go to software upgrade and download and install 8.4.1
  4. After you are on 8.4.1 go to General > Reset > Erase all settings and content
You may now go and jailbreak with etas0n JB for an Untethered Jailbreak
