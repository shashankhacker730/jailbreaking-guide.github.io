JailbreakMe 2.0: Fix "untrusted server certificate"

This method only supports devices jailbroken with the JailbreakMe 2.0 guide on this website.
For this guide you will need the following:
  • Wi-Fi Access (WPA/WPA2 with password works too)
  • Device jailbroken with the JailbreakMe 2.0 Guide on this website
  • Charles Proxy 
Step 1: Charles
  1. Open Settings > Wi-Fi > Press the Info button next to your Wi-Fi network
  2. Scroll to the bottom, and select Manual for HTTP Proxy
  3. Visit whatismyip.com on your computer and put your Local IP in for the Server field on your device, then put 8888 into the Port field.
  4. Open Safari, and visit charlesproxy.com/getssl and install the certificate.
Your computer will fail to load websites while Charles is open. To fix this, in the menu go to Help > SSL Proxying > Install Charles Root Certificate. Follow the steps carefully.
Charles only runs for 30 minutes, and you must be on your local network to use it, so open Charles and repeat this step before using Cydia.
You may see a message on your computer asking if you would like to allow your device to connect. Press Allow.
Step 2: Fix "untrusted server certificate"
  1. Open Cydia, then do a Complete Upgrade
  2. Add the repo "cydia.invoxiplaygames.uk"
  3. Download Cydia HTTPatch from that repo
  4. Reboot your device
  5. Delete Charles, as you will never need it again
  6. Go to your Wi-Fi settings and set Proxy to None (essentially reverting Step 1)
If this fails, try connecting your computer to the Internet via Ethernet instead of WiFi, or try a Windows PC.
