iTunes Backups

Du brauchst einen Mac oder einen Windows PC um fortzufahren
Für diese Guide brauchst du folgende sachen;
  • iOS Gerät
  • Lade & Sync Kabel für dein iOS Gerät.
  • Mac oder Windows PC mit genug Speicherplatz für ein Backup.
  • iTunes für Windows Benutzer
  • Windows Benutzer müssen eventuell iTunes vom microsoft laden herunterladen.
macOS Catalina 10.15.X benutzer müssen die Finder app benutzen um ein Backup zu machen und es wieder herzustellen.
Step 1: Ein Backup
  1. Auf einem Mac, öffne iTunes
    2. iTunes ist schon preinstalliert auf Macs und ist entweder im Dock oder im Launchpad. Windows benutzer können iTunes hier herunterladen.
  2. Verbinde dein iOS Gerät mit dem Lade & Sync Kabel mit deinem Mac oder Windows PC
    3. Du braucht eventuell ein USB-C zu Lightning Cable für neuere Macs.
    Falls dein Gerät für mehr als eine Stunde gesperrt ist,wirst du gefragt dein iOS Gerät zu entsperren (iOS 12/13).
  3. Navigiere zu deinem iOS Gerät
  4. Überprüfe die Box "Entschlüssele (iPad/iPhone/iPod Touch) Backup". Du wirst aufgefordert ein Passwort einzugeben.
    5. Falls du das Passwort vergisst, kannst du dein iOS Gerät mit dem Backup nicht wiederherstellen.
  5. Drücke Jetzt sichern. Dies könnte eine Weile dauern.
Nachdem das backup fertig ist, kannst du weitermachen bei der Jailbreaking Guide.
